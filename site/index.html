Dinoqode

A kid-friendly system for controlling SONOS® with QR codes.

Get Started

Open Source

This project is relased under the MIT License.

QR-Code Recognition

With command cards and song/album cards it's possible to control any SONOS® speaker in the same network. A camera recognize the QR Code and send commands to the node-sonos-http-api.

LEGO® casing

The Raspberry PI is put into a LEGO® case. Multiple LEGO® building kits can be used. BE CREATIVE!

Supports major music services

The music services Amazon Music, Apple Music, Spotify and Napster (incl. Aldi Life Music) are supported.

Powered by Raspberry PI.

Dinoqode is powered by an Raspberry PI and some custom software. It uses the Raspberry PI camera (V2.1 or V1.3) to recognize QR Code cards and control / play songs/albums on a SONOS® speaker in the same network. The project node-sonos-http-api is used to control the SONOS® speaker.

This project is inspired by qrocodile

Hardware list

  • Rasperry Pi 3 Model B+
  • Raspberry Pi V2.1 Camera or Raspberry Pi V1.3 Camera
  • Raspberry Pi 3 power adapter
  • Pimoroni Blinkt! LED
  • Pi-Blox LEGO Case
  • SanDisk microSDHC
  • LEGO Creator Dinosaur
  • LEGO Classic Box

Software list
